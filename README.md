# 🎙️ AI 语音助手

基于 DeepSeek 大模型的智能语音对话助手，支持中英文语音输入输出。

## ✨ 功能特性

- 🤖 接入 DeepSeek-V3，流式响应，回复快速
- 🎤 浏览器原生语音识别，支持中英文切换
- 🔊 语音朗读回复内容，支持暂停 / 继续
- 💬 对话历史本地持久化
- 📋 一键复制消息内容
- 📱 移动端友好，响应式布局

## 🚀 在线体验

👉 [https://ai-voice-chat-v3th.vercel.app](https://ai-voice-chat-v3th.vercel.app)

## 🛠️ 技术栈

- **框架**：Vue 3 + TypeScript + Vite
- **UI**：自定义组件 + Vant
- **AI 对话**：DeepSeek API（流式输出）
- **语音识别**：Web Speech API
- **语音合成**：SpeechSynthesis API
- **部署**：Vercel
